# SplitTrading — 분할매매 계산기

BitSplit 스타일의 암호화폐 분할매매 계산기 웹앱입니다. 빌드 도구나 외부 라이브러리 없이 순수 HTML/CSS/JavaScript로 만들어져 오프라인에서도 동작합니다.

## 기능

| 탭 | 설명 |
|---|---|
| **분할매수** | 총 투자금을 최고가~최저가 구간에 나눠 매수할 때 각 주문의 가격·금액·수량과 평균 매수단가를 계산 |
| **분할매도** | 보유 수량을 최저~최고 목표가 구간에 나눠 매도할 때의 주문표와 평균 매도단가, (평단 입력 시) 수익률 계산 |
| **물타기** | 추가 매수 시 새 평균단가, 평단 하락률, 본전까지 필요한 상승률 계산 |
| **수익 계산** | 매수가·매도가·수량·수수료율로 실현 손익과 수익률 계산 |

분할매수·매도는 **균등 분배**와 **가중 분배**(매수: 낮은 가격일수록 큰 금액, 매도: 높은 가격일수록 많은 수량)를 지원합니다.

## 로컬 실행

정적 사이트이므로 아무 HTTP 서버로 열면 됩니다.

```bash
python3 -m http.server
# 브라우저에서 http://localhost:8000 접속
```

또는 `npm run serve`.

> ES 모듈을 사용하므로 `file://`로 직접 열지 말고 HTTP 서버를 통해 접속하세요.

## 테스트

계산 로직(`js/calc.js`)은 DOM에 의존하지 않는 순수 함수 모듈이며, Node.js 내장 테스트 러너로 검증합니다.

```bash
npm test
```

## 계산 공식 요약

- **주문 가격**: 구간을 N개로 선형 등분 — `가격ᵢ = 시작가 - (시작가 - 끝가) × i/(N-1)`
- **가중 분배**: i번째 주문 가중치 `i+1`, 배분 비율 `= (i+1) / (1+2+…+N)`
- **평균 매수단가** `= 총 매수금액 ÷ 총 매수수량`
- **물타기 새 평단** `= (기존평단×보유수량 + 추가매수금액) ÷ 총수량`
- **본전 필요 상승률** `= (새평단 ÷ 추가매수가 − 1) × 100`
- **실현 손익** `= 매도가×수량×(1−수수료) − 매수가×수량×(1+수수료)` (수수료 기본 0.05%)

